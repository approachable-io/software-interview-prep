Memory Efficient Doubly Linked List

More whiteboard interview questions

Question:

Write a pseudo code or algorithm to implement a doubly linked list with only one pointer.

Approach:

Storing the XOR of the previous and next pointer, the current pointer of the head will the XOR of the new and the current node. So if we do XOR of current pointer and the previous node we get the next node.
