Briefly describe structs and classes in C#. How are they similar, and how are they different?

More technical interview questions

Common Traits

  • Both are compound data types
  • Both can contain methods
  • Both can implement interfaces

Unique Class Traits

  • Support Inheritance in addition to interfaces
  • Are passed and used as references
  • Can be null

Unique Struct Traits

  • Do not support inheritance
  • Are value types, and passed by value
  • Cannot have a null reference, unless encapsulated in a nullable
