What is Angular Bazel?

The @angular/bazel package provides a builder that allows Angular CLI to use Bazel as the build tool. Now when you use Angular CLI build commands such as ng build and ng serve , Bazel is used behind the scenes. Outputs from Bazel appear in the dist/bin folder. The command-line output includes extra logging from Bazel.
